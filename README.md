# Web Data Extractor Pro

A powerful web-based tool to extract website data in bulk.

## Features

- ✅ Extract up to 200 URLs at once
- ✅ Get Page Titles
- ✅ Get Meta Descriptions
- ✅ Extract Email Addresses
- ✅ Extract Phone Numbers
- ✅ Detect Site Age
- ✅ Download as CSV/Excel
- ✅ Mobile Responsive
- ✅ SEO Friendly
- ✅ Blog with Articles
